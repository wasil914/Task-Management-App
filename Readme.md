<h2>⚙️ Setup Instructions</h2>

<pre>
# Clone the repo
git clone https://github.com/wasil914/Task-Management-App.git

# FRONTEND
cd client
npm install
npm run dev

# BACKEND
cd ../server
npm install
npm run dev
</pre>

Make sure to create a <code>.env</code> file in <code>/server</code> with the following:
<pre>
MONGO_URI=your_mongodb_connection_string
PORT=5000
</pre>
